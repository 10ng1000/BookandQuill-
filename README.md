# BookandQuill
游戏Minecraft的辅助记录程序，大一qt学习项目
Minecraft中（特别是原版生存）经常需要记录坐标和计算坐标（测距），原版游戏的记录不太方便，计算也需要手算，为此开发此程序以实现上述功能。除这些核心功能外，还加入了一些视觉和实用的功能如快捷打开游戏，显示游戏截图等。
